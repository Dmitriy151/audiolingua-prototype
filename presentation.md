# AudioLingua — Презентация проекта

## Проблема

📚 **Изучающие языки сталкиваются с трудностями:**
- Аудиокниги без перевода сложно понимать
- Текстовые переводы не развивают аудирование
- Ручное создание материалов занимает часы

## Решение

🎧 **AudioLingua автоматически создаёт аудиоуроки:**

1. **Загружаете любой текст** (рассказ, статью, главу книги)
2. **Получаете аудио с переводом:**
   - Предложение на английском
   - Перевод на русский
   - Пояснение сложных выражений
3. **Слушаете в удобном темпе** с синхронной подсветкой

## Уникальность

✨ **Никто не делает так:**

| Сервис | Свой текст | По-предложный перевод | Пояснения |
|--------|-----------|---------------------|-----------|
| Beelinguapp | ❌ | ✅ | ✅ |
| Speechify | ✅ | ❌ | ❌ |
| **AudioLingua** | ✅ | ✅ | ✅ |

## Целевая аудитория

## MVP Функционал

🚀 **Первая версия (EN→RU):**
- ✅ Загрузка текста (TXT/PDF/URL)
- ✅ Выбор голосов и скорости

## Технологический стек

🛠️ **MVP:**
- Frontend: React + Vite
- Backend: Python FastAPI
- TTS: Yandex SpeechKit / OpenAI
- Translation: DeepL API
- DB: Supabase (PostgreSQL)
