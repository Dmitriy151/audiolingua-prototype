# 🎧 AudioLingua — Генератор аудиокниг с переводом

> Сервис создания аудиоуроков с по-предложным переводом и пояснениями

🌐 **Живая демо-версия:** [https://Dmitriy151.github.io/audiolingua-prototype/](https://Dmitriy151.github.io/audiolingua-prototype/)

> Сервис создания структурированных аудиоуроков из любого текста с настраиваемым размером сегментов, переводом и пояснениями.

## 📖 О проекте
AudioLingua закрывает разрыв между чтением и аудированием. Пользователь загружает главу книги или статью, а сервис автоматически генерирует структурированный аудиофайл по заданной схеме:
1. 🔊 Чтение N предложений на языке оригинала (EN/DE/FR/ES)
2. 🌍 Перевод этих же предложений на русский язык
3. 💡 Текстовые или аудио-пояснения сложных выражений и идиом
4. 🔗 Серверное склеивание сегментов в единый MP3-файл главы с возможностью дальнейшей сборки в полноценную аудиокнигу.

## ✨ Ключевые возможности (Актуальная версия)
- 📥 **Загрузка текста:** вставка вручную, drag-and-drop файлов или парсинг по URL.
- ⚙️ **Гибкие настройки чанков:** выбор количества предложений в одном сегменте (от 1 до 10) перед переключением на перевод.
- ⚡ **Реальная генерация:** интеграция с Yandex Translate API и Yandex SpeechKit (TTS) через надёжный Python/FastAPI бэкенд.
- 🔗 **Серверная склейка (FFmpeg):** автоматическое объединение десятков мелких аудиофрагментов в один бесшовный `chapter.mp3` без потери качества.
- 📚 **Сборка аудиокниг:** эндпоинт и интерфейс для объединения нескольких сгенерированных глав в один файл `audiobook.mp3`.
- 🎧 **Интерактивный плеер:** синхронная подсветка текущего текстового сегмента, управление скоростью воспроизведения (`playbackRate`) и навигация по главам.
- 📥 **Прямой экспорт:** мгновенное скачивание готовой скомпилированной главы в формате MP3.

## 🚀 Как протестировать
1. Откройте [демо-версию](https://Dmitriy151.github.io/audiolingua-prototype/) в браузере
2. На шаге «Загрузка» вставьте английский текст или используйте демо-текст по умолчанию
3. Нажмите **«Далее: Настройки»** → выберите голоса, скорость и опции
4. Нажмите **«Сгенерировать аудиоурок»** → дождитесь симуляции процесса генерации
5. В плеере управляйте воспроизведением, переключайте предложения, открывайте пояснения и просматривайте словарь

> ⚠️ **Примечание:** в текущей версии аудио и тайм-коды симулируются на клиенте для демонстрации UX. В продакшене генерация будет происходить на сервере через API TTS и ML-перевода.
