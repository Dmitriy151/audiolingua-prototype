# 🎧 AudioLingua — Интерактивный прототип

**Сервис создания аудиоуроков с по-предложным переводом и пояснениями**

## 🌐 Демо

Откройте [index.html](index.html) в браузере или используйте GitHub Pages:
👉 [https://yourusername.github.io/audiolingua-prototype](https://yourusername.github.io/audiolingua-prototype)

## 🚀 Как запустить

### Вариант 1: Локально
1. Скачайте файлы проекта
2. Откройте `index.html` в любом браузере
3. Готово!

### Вариант 2: GitHub Pages
1. Запушьте репозиторий на GitHub
2. Settings → Pages → Source: main branch
3. Через 1-2 минуты сайт будет доступен по ссылке

## 📋 Что можно протестировать

✅ **Загрузка текста** — вставьте текст, загрузите файл или укажите URL  
✅ **Настройки** — выберите голоса, скорость, опции  
✅ **Генерация** — симуляция процесса создания аудиоурока  
✅ **Плеер** — прослушивание с синхронной подсветкой предложений  
✅ **Пояснения** — кликайте на ℹ️ чтобы показать/скрыть объяснения  
✅ **Словарь** — просмотр слов из урока  
✅ **Экспорт** — выбор форматов для скачивания  

## 🎨 Технологии

- HTML5 + CSS3 (Flexbox, Grid, CSS Variables)
- Vanilla JavaScript (ES6+)
- Адаптивный дизайн (mobile-first)
- Google Fonts (Inter)
