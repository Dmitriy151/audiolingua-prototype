# 🎧 AudioLingua — Интерактивный прототип

> Сервис создания аудиоуроков с по-предложным переводом и пояснениями

🌐 **Живая демо-версия:** [https://Dmitriy151.github.io/audiolingua-prototype/](https://Dmitriy151.github.io/audiolingua-prototype/)

## 📖 О проекте
AudioLingua закрывает разрыв между чтением и аудированием при изучении иностранных языков. Пользователь загружает любой текст (рассказ, статью, главу книги), а сервис автоматически генерирует структурированный аудиоурок:
1. 🔊 Предложение на языке оригинала
2. 🌍 Перевод на русский язык
3. 💡 Пояснение сложных выражений, идиом и грамматических нюансов

## ✨ Возможности прототипа
- 📥 **Загрузка текста:** вставка вручную, drag-and-drop файлов (TXT/PDF/EPUB), парсинг по URL
- ⚙️ **Гибкие настройки:** выбор голосов (EN/RU), регулировка скорости, включение/отключение пояснений
- 🎧 **Синхронный плеер:** аудио воспроизводится в удобном темпе с синхронной подсветкой текущего предложения в тексте
- 📖 **Интерактивные пояснения:** клик по иконке ℹ️ раскрывает объяснение идиом, фразовых глаголов или культурных отсылок
- 📚 **Автоматический словарь:** сложные слова из текста сохраняются в личный список для повторения
- 📥 **Экспорт:** возможность скачать MP3, SRT-субтитры или PDF с параллельным текстом (демо-режим)

## 🚀 Как протестировать
1. Откройте [демо-версию](https://Dmitriy151.github.io/audiolingua-prototype/) в браузере
2. На шаге «Загрузка» вставьте английский текст или используйте демо-текст по умолчанию
3. Нажмите **«Далее: Настройки»** → выберите голоса, скорость и опции
4. Нажмите **«Сгенерировать аудиоурок»** → дождитесь симуляции процесса генерации
5. В плеере управляйте воспроизведением, переключайте предложения, открывайте пояснения и просматривайте словарь

> ⚠️ **Примечание:** в текущей версии аудио и тайм-коды симулируются на клиенте для демонстрации UX. В продакшене генерация будет происходить на сервере через API TTS и ML-перевода.

## 🛠️ Технологический стек
- **Frontend:** HTML5, CSS3 (Flexbox/Grid, CSS Variables), Vanilla JS (ES6+)
- **Аудио-логика:** Howler.js (симуляция синхронизации и тайм-кодов)
- **Дизайн:** Google Fonts (Inter), адаптивная вёрстка (mobile-first)
- **Хостинг:** GitHub Pages (автоматический деплой из ветки `main`)
